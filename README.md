### 导入
> * 手动导入：
>
>> * **SwpDeviceIpUtils** 文件夹导入项目中。
>> * **`#import "SwpDeviceIpUtils"`**


> -------

> * CocoaPods 导入:
>
>> * **pod search SwpDeviceIpUtils**
>> * **pod 'SwpDeviceIpUtils'**
>> * **pod insterall**
>> * **`#import <SwpDeviceIpUtils/SwpDeviceIpUtils.h>`**
